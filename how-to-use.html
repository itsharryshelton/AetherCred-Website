How to Use AetherCred

A guide to setting up and running the AetherCred security posture dashboard.

Overview

AetherCred is a PowerShell and HTML-based tool designed to provide a clear and actionable overview of your Microsoft Entra ID security posture. It gathers detailed information about users, their MFA status, roles, and other security-related attributes, then presents it in an intuitive, searchable, and exportable web dashboard.

This tool allows for quick and easy report generating that can be easily shared with non-technical customers to help them better understand their security posture within their Entra estate.

Key Features

  • Comprehensive User Analysis: Gathers data on all users, including their enabled status, creation date, last sign-in, and password policies.
  • Detailed MFA Reporting: Differentiates between Modern MFA (Passwordless, Authenticator App, FIDO2) and Legacy MFA (SMS, Voice, TOTP).
  • Security Scoring: Assigns each user a security score from 0-100 based on critical risk factors.
  • Risk Flagging: Automatically flags potential security risks such as "MFA Not Registered," "Password Expiry Enabled," and "Never Signed In."
  • Privileged Role Identification: Highlights users with high-impact administrative roles.
  • Interactive Web Dashboard: Features a clean, dark-mode interface with at-a-glance summary cards and a searchable, sortable data table.
  • Multiple Export Options: Generate a branded PDF report directly from the dashboard or use the automatically generated CSV file for analysis in other tools.

Prerequisites

  1. PowerShell: Version 5.1 or higher.
  2. Microsoft Graph PowerShell SDK: If not installed, run this command in PowerShell as an administrator: 
    Install-Module Microsoft.Graph -Scope AllUsers
  3. Required Permissions: The user running the script needs consent for the following Microsoft Graph API scopes: `Application.ReadWrite.All`, `Directory.Read.All`, `User.Read.All`, `UserAuthenticationMethod.Read.All`, `AuditLog.Read.All`, `Reports.Read.All`.

Setup & Usage

  1. Download Files: Place `AetherCred.ps1`, `AetherCred-Dashboard.html`, `AetherCredLogoCloud.png`, and `favicon.ico` into the same folder.
  2. Run the Script: Open PowerShell, navigate to the folder, and execute the script: 
    .\AetherCred.ps1
  3. Authenticate: A Microsoft Graph sign-in window will appear. Sign in with a privileged account. You may be asked to consent to the required API permissions.
  4. View the Report: Once finished, the script automatically generates the data files and launches the HTML dashboard in your default browser.

How It Works

The project is split into two main parts:

  • Data Collection (PowerShell): The AetherCred.ps1 script connects to the Microsoft Graph API to fetch all relevant data. It processes this information, calculates scores, and then exports the final dataset into a .js file for the web dashboard and a .csv file for external use.
  • Data Visualisation (HTML/JavaScript): The AetherCred-Dashboard.html file uses the generated .js file as its data source. It uses JavaScript to dynamically build the summary cards and the main data table, providing an interactive user experience without needing a web server.

Customisation

PowerShell Script (`AetherCred.ps1`)

You can customise the Application details by modifying the variables at the top of the script:

# SCRIPT CONFIGURATION
$AppName = "AetherCred"
$LogoFileName = "AetherCredLogoCloud.png"
$HomePageURL = "https://github.com/itsharryshelton"

Dashboard (`AetherCred-Dashboard.html`)

To ensure your logo appears correctly on PDF exports, you must update the hidden `` tag in the HTML file.

  1. Convert your logo image to a Base64 string using a free online tool.
  2. Open AetherCred-Dashboard.html in a text editor.
  3. Find the line: <img id="logo-for-pdf" src="" ... >
  4. Paste your Base64 string inside the src="" attribute.

Roadmap

Future enhancements are planned to make AetherCred even more valuable. Contributions and suggestions are always welcome!

  • Enhanced Scoring System: Introduce more granular scoring based on MFA types and user risk levels from Entra ID Protection.
  • Historical Data & Trend Analysis: Add a "Trends" tab to visualise security posture improvements over time.
  • Deeper Risk Analysis: Integrate with Entra ID Protection to flag risky users and check for stale privileged accounts.
  • Improved Dashboard Interactivity: Add advanced filtering, column sorting, and a light/dark mode toggle.
